# Aegis Emergency AI

Aegis Emergency AI is an AI-powered emergency response platform that provides real-time assistance, emergency reporting, and smart crisis management support.

---

## Features

- Emergency reporting system
- AI-powered assistance
- Crisis map visualization
- Volunteer management
- Admin dashboard
- User authentication
- Analytics dashboard

---

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## Installation

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
